The ZCA Bootstrap template includes a Modal feature you can use on the checkout_payment page for the CVV Help Popup window for payment modules.

To use the Bootstrap Modal feature, find the modules payment module in /includes/modules/payment/

Each module shows a link to the CVV Help Popup window:
a href="javascript:popupWindow('' . zen_href_link(FILENAME_POPUP_CVV_HELP) . '')"

This link needs to be replaced with:
a data-toggle="modal" href="#cvvHelpModal"

That's it, the Bootstrap Modal feature will be activated!
